Vocabulary list for Introduction

Brooke Anderson
Last updated on Jan 13, 2020 2 min read vocabulary, Introduction
exploratory data analysis (EDA) A method of statistical data analysis that uses the data themselves to decide how to conduct the statistical analysis, often leveraging simple tools for plotting data.
confirmatory data analysis (CDA) A method of statistical data analysis that uses robust inferential methods that do not rely on complex assumptions to reach scientific conclusions.
the large-p small-n problem A description of analyses where more model parameters would be fit than there are observations in the data.
empirical Bayes An analysis method that does not try to learn the parameters associated with each feature from scratch, but rather uses the fact that some or all of them will be similar, or even the same, across all features, or across groups of related features.
simulations Artificially created data based on random number generation that can be used to model random events.
generative models Simple models describing the probabilistic process by which the data of interest came about.
